Projekt pro TNPW2 & OWE
=============
###MongoDB, Express, React + Redux, Nodejs
###Spuštění DB
```shell
cd C:\to_mongoDB_location
\bin\mongod.exe --dbpath=mongodb-data
```


###Spuštění BE
```shell
cd server
npm run start
```


###Spuštění FE
```shell
cd client
npm start
```
