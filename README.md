Projekt pro TNPW2 & OWE
=============

###Spuštění DB
```shell
cd C:\Users\simis\Desktop\TNPW2\mongoDB
mongodb\bin\mongod.exe --dbpath=mongodb-data
```


###Spuštění BE
```shell
npm run server
```


###Spuštění FE
```shell
cd client
npm start
```

###Spuštění FE & BE dohromady
```shell
npm start
```
